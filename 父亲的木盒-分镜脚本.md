# 示例脚本：《父亲的木盒》第一人称 POV 分镜

> 用 `pov-script` 技能，对一篇第一人称亲情小说（父亲失智、思念亡妻）实跑产出的完整脚本。
> 画幅：竖屏 9:16｜目标：即梦 Seedance 2.0（单条 4-15 秒、带声音）｜6 场 / 26 段 / 约 285 秒。
> 演示要点：长剧本不被「60 秒 / 起承转合四幕」砍断，按戏剧节拍自适应分段，双高潮结构完整保留，全片过审。

# 《剧本解析》

- **题材类型**：现实亲情 / 记忆与失智（暖中带苦、苦中回甘）
- **POV 主体**："我"= 方远，34 岁。镜头是"我"的主观所见，主体永远是**父亲**；"我"仅以手、手臂、背影、倒影入画，台词作主观画外音。
- **核心人物表**
  - **父亲**（约 62 岁）：本片唯一持续出场的画面主体，沉默隐忍，因失智活在对亡妻的思念里。承担全部情绪重量。
  - **母亲**（约 43 岁，已故三年）：**不真实出场**，仅存在于老照片、台历字迹、父亲的呼唤与一把空藤椅 / 空碗的"缺席"中。
  - **我 / 方远**（34 岁）：观察者与情感出口，克制、心软，最终选择不再戳破父亲的幻觉。
- **时空设定**：现代（模糊为"近些年"，过审）。两个空间——**城里·我家**（明亮、整洁、陌生）与**老家·老房子**（陈旧、落灰、有记忆）。季节由秋入冬（棉袄意象）。地域默认**北方小城**气质，不用强方言。
- **核心情绪与主题**：有些爱固执而绵长，不需要真实存在，只存在于念念不忘里，就已足够。基调：克制的疼。
- **情绪曲线与高潮点**：平稳铺垫（接来→古怪初现）→ 两次中度刺痛（红烧肉、深夜找妈）→ **第一高潮**（客厅"你妈来电话了"+ 开盒三件遗物，第 5 场）→ 回落决定 → **情感落点高潮**（老房子"老伴，我回来了"，第 6 场）→ 留白收尾。双高潮结构。
- **关键意象 / 符号道具**：① 旧木盒 ② 锈钥匙 ③ 老式电话本（"老伴"二字）④ 停摆台历（"明天买白菜"）⑤ 泛黄合影（棉袄）⑥ 空藤椅 / 空碗（母亲的"缺席在场"）。
- **改编假设**：竖屏 9:16；不用强方言；目标总时长 ~4.7 分钟 / 26 段；默认无字幕、无 BGM，保留对白 + 环境声。

# 《分镜脚本》（6 场 / 26 段 / 约 285 秒）

## 第 1 场　城里·我家（接来 / 木盒）

【第 1 段｜场：我家·玄关｜时长 9s｜出场角色：父亲（一致性锚：国字长脸、招风耳、花白浓眉、微驼努力挺直）】
镜头景别：中近景
镜头视角：第一人称 POV，"我"站在玄关看父亲；我的手扶着门框入画
画面内容：父亲——国字长脸、招风耳、花白浓眉的老人，深灰旧开衫，刚进门，脚边只有一只旧帆布包；他却双手把一只磨损的旧木盒紧紧抱在胸前，低头看了盒子一眼，又抬眼怯怯地看向镜头（看我），脚步在陌生的新家门口迟疑地顿住；身后是明亮陌生的城市公寓走廊
台词与声音：我(OS，温和)：「爸，进来吧，以后这就是家。」父亲(低声)：「哎……东西不多，这个盒子……我自己拿着。」
镜头运镜：轻微手持起伏的固定 → 缓缓下摇到他怀里的木盒 → 停
光影色调：黄昏暖光从走廊斜入，低饱和暖调，米黄+暖灰
氛围质感：陌生、迟疑、隐忍
环境声设计：环境音=走廊回声、远处电梯提示音；动作音=帆布包落地的闷响、布鞋蹭地声
附加：浅景深，背景虚化；胶片颗粒、写实质感；毛衣袖口起球、木盒掉漆磨损；无字幕、无 BGM
