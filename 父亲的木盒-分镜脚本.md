# 《台词白名单》（全文仅此 9 处直接引语，分镜台词只能用这些）

- **W1 父亲**：「你妈想吃红烧肉了，我正在做。」
- **W2 我**：「爸，妈已经走了三年了。」
- **W3 父亲**：「你妈不见了，她刚才还在屋里，我就去上了个厕所，人就不见了。」
- **W4 父亲**：「远儿，咱们得回老房子了。刚才你妈来电话了，说她要回来了。咱们搬家了，也不告诉她一声，她回家找不到人，该着急了。」
- **W5 我**：「爸……」

- **W8 我**：「爸，你那个盒子里，到底装的什么？」

# 《剧本解析》

- **题材类型**：现实亲情 / 记忆与失智（暖中带苦、苦中回甘）
- **POV 主体**："我"= 方远，34 岁。镜头是"我"的主观所见，主体永远是**父亲**；"我"仅以手、手臂、背影、倒影入画，台词作主观画外音。
- **核心人物表**
  - **父亲**（约 62 岁）：本片唯一持续出场的画面主体，沉默隐忍，因失智活在对亡妻的思念里。承担全部情绪重量。
  - **母亲**（约 43 岁，已故三年）：**不真实出场**，仅存在于老照片、台历字迹、父亲的呼唤与一把空藤椅 / 空碗的"缺席"中。
  - **我 / 方远**（34 岁）：观察者与情感出口，克制、心软，最终选择不再戳破父亲的幻觉。
- **时空设定**：现代（模糊为"近些年"，过审）。两个空间——**城里·我家**与**老家·老房子**。季节由秋入冬（棉袄意象）。地域默认北方小城气质，不用强方言。
- **核心情绪与主题**：有些爱固执而绵长，不需要真实存在，只存在于念念不忘里，就已足够。
- **情绪曲线与高潮点**：平稳铺垫 → 两次中度刺痛（红烧肉、深夜找妈）→ **第一高潮**（客厅 W4/W6 + 开盒三件遗物，第 5 场）→ 回落决定 → **情感落点高潮**（老房子 W9，第 6 场）→ 留白收尾。双高潮结构。
- **关键意象 / 符号道具**：旧木盒、锈钥匙、老式电话本（"老伴"二字）、停摆台历（"明天买白菜"）、泛黄合影（棉袄）、空藤椅 / 空碗。
- **改编假设**：竖屏 9:16；不用强方言；目标 ~4.7 分钟 / 26 段；默认无字幕、无 BGM，保留对白 + 环境声。
