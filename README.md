# pov-script

把**任意剧本或小说**改编成「第一人称 POV」分镜脚本的 Claude Code / Codex 技能，输出可直接喂给**即梦 Seedance 2.0**（单条 4-15 秒、带声音）的提示词。

不绑定题材（亲情 / 悬疑 / 爱情 / 年代 / 恐怖 / 历史 / 武侠 / 科幻 …）。**段数与总时长由剧本自身的戏剧节拍决定**，不套固定幕数、不限总时长——长剧本就是几分钟、几十段，绝不为凑"起承转合 60 秒"删改剧情。

## 目录结构

```
skills/pov-script/
├── SKILL.md                      # 主工作流：读懂剧本 → 角色一致性 → 自适应分段 → 分镜
└── references/
    ├── consistency-and-4view.md  # 角色基因卡 + 多年龄段 4 视图 + 跨图一致性硬约束
    ├── era-and-dialect.md        # 年代 / 地域 / 方言工具箱（可扩展）
    └── seedance-compliance.md    # 即梦 Seedance 2.0 过审规避规则

examples/
└── father-music-box-pov-storyboard.md   # 完整示例：《父亲的木盒》6 场 / 26 段 / ~4.7 分钟
```

## 核心设计

1. **剧本驱动，不是模板驱动**：先吃透剧本（题材 / POV 主体 / 人物表 / 时空 / 戏剧节拍 / 情绪曲线 / 高潮点 / 关键意象），再让结构自然涌现。
2. **第一人称到底**：明确"我"是谁，画面是"我"的主观所见，"我"仅手 / 局部身体 / 背影 / 倒影入画。
3. **跨角色 / 跨年龄一致性**：对每个重要角色建"角色基因卡"，有年龄跨度的出多年龄段 4 视图，并复述核心特征锁定一致性；推荐 4 视图出图 → 取首帧 → 图生视频，强化一致性。
4. **对齐 Seedance 2.0 有声能力**：对白与环境声直接写进提示词，不再当纯后期。
5. **过审优先**：所有描写先过一遍即梦审核规避规则（敏感身体 / 动作 / 时间地点 / 情绪 / 画面词的替代写法）。

## 输出结构（三段式）

1. 《剧本解析》
2. 《角色一致性设定》：N 张角色基因卡 + 必要的多年龄段 4 视图
3. 《分镜脚本》：N 段分镜提示词（每段 8 字段：景别 / 视角 / 画面内容 / 台词与声音 / 运镜 / 光影色调 / 氛围质感 / 环境声 / 附加）

## 使用方式

### Claude Code
把 `skills/pov-script` 放到 `.claude/skills/` 下（项目级或用户级 `~/.claude/skills/`），在该工作区用 `/pov-script` 调用，或直接描述需求触发。提供剧本/小说原文即可。

### 即梦 Seedance 2.0
1. 先用《角色一致性设定》里的 4 视图提示词出**角色设定图**；
2. 取设定图作参考 / 首帧，逐段把《分镜脚本》喂给 Seedance（图生视频，一致性更强）；
3. 每段对应一个 4-15 秒有声片段，按场顺序拼接成片。

详见 `examples/father-music-box-pov-storyboard.md` 的完整产出范例。
