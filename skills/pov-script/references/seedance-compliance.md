# 即梦 Seedance 2.0 过审规则（必守，避免误伤审核）

任意题材在输出前都要逐段过一遍本表。目的是让正常的剧情（尤其情绪戏、年代戏、冲突戏）不被审核误判，靠台词 + 环境声 + 柔和动作 + 道具暗示传递强度，而非极端身体细节与高强度动作词。

## 身体描写规避

- 不要写：痣、泪痣、胎记、疤痕、伤口
- 不要写：喉结滚动、指关节泛白、青筋暴起、血丝
- 不要写：发抖、颤抖、抽搐、抽泣到抽搐
- 不要用网络化人脸描述：狗狗眼、死鱼眼、八字眉等
- 改用：眼角微红、眼神低垂、嘴唇微抿、手指轻轻捏紧、肩膀轻颤、低头不语

## 动作描写规避

- 不要用："猛地""急速""剧烈""撕扯""抓打""暴怒"等高强度词
- 不要用运镜词："急推""急拉""急摇""快切"
- 改用："缓缓""慢慢""自然""轻轻""舒缓推近""自然摇过"

## 时间地点规避

- 不写敏感年份（如 2008、1989 等），用"2000 年代末""90 年代初"模糊化
- 大城市具体地名慎用，用"老家""学校""外地""城里"代替
- "绿皮火车"改"老式火车"

## 情绪场景规避

- 哭戏不写"嚎啕""崩溃""失声痛哭"，改"眼角微红""轻轻吸鼻""低头不语"
- 离别场景不叠加"留守""失去""再也不见"等暗示
- 病 / 亡场景慎做，改用"老去""离开""出远门"暗示（如阿尔茨海默、临终等剧情，用恍惚、念叨、空椅子、旧物等意象表达，不写病理细节）

## 画面内容规避

- 不出现刀具、明火、酒瓶细节、烟雾、医院病床特写、灵堂等
- 婚礼场景中"红盖头"OK，但不要写"哭嫁""跪拜亡父"等
- 暴力 / 惊悚 / 恐怖题材：用光影、声音、留白、暗示营造，不展示血腥、凶器、尸体细节

## 替代思路

靠台词 + 环境音 + 柔和动作 + 道具暗示传递情绪强度，少用极端身体细节和高强度动作词。情绪越重，越靠意象与留白，而不是越靠直白的生理描写——这样既过审，也更高级。
