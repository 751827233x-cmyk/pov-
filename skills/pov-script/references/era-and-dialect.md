# 题材自适应工具箱：年代符号 / 地域符号 / 方言

这是一个**可扩展样例库**，不是封闭清单。剧本涉及库里没有的年代、地域、题材（如民国、古装武侠、未来科幻、海外、校园、职场、悬疑等）时，按同样思路自行补足该时空下合理、写实、可过审的视觉符号与口语。目标是用 2-3 件强符号道具 + 写实环境 + 自然口语把观众瞬间带进那个时空。

## 年代符号样例（按出生 / 事件年代调用）

- 1985-1990：筒子楼 / 家属院、蜂窝煤、二八大杠、铁皮玩具、的确良、搪瓷缸、虎头棉鞋、黑白电视机、冰棍儿、收音机里的豫剧 / 评书
- 1990-1995：水泥小院、小霸王学习机、汽水瓶、动画城、奥特曼卡片、塑料发卡
- 2000 年代（成年节点）：老火车站 / 长途汽车站、蛇皮袋、红蓝白编织袋、回力鞋、冰红茶、发条手机
- 2010 年代（成年节点）：智能手机、动车站、行李箱、修身西装、新中式婚俗
- 2020 年代：短视频、外卖箱、无线耳机、新能源车、直播补光灯
- 需要更早 / 更晚 / 非大陆背景时，自行补足该时代写实物件（如民国旗袍黄包车、古代油灯竹简、未来感全息界面等），保持写实与过审。

## 地域符号样例

- 河南 / 北方：红砖、暖气片、年画胖娃娃、唢呐、平原麦田
- 四川 / 西南：青瓦、雾气、竹林、辣条、麻将声、盆地阴天
- 东北：大花棉袄、二人转、铁锅炖、雪地、火炕
- 江南 / 水乡：白墙黛瓦、石桥、乌篷船、梅雨、弄堂
- 其它地域自行补足代表性建筑、气候、饮食、声音景观。

## 方言模板样例

- 河南话：恁、中、咋、哩、乖 / 娃、走类
- 四川话：你娃、晓得不、要得、摆龙门阵、咋个、哈 / 嘛
- 东北话：嘎哈、咋整、贼、稀罕、瞅啥、整两口
- 粤语、上海话、陕西话等按需补足代表性词汇与语气助词
- 不用方言时：用"啊 / 哎 / 哦 / 嘛"等语气词强化口语感，避免书面腔

## 调用原则

- 符号服务情绪与情节，不堆砌：每段挑 2-3 件最能定调的即可。
- 年代 / 地域 / 方言要内部自洽：人物年龄、场景物件、口音必须对得上时间线。
- 一切符号仍需过 `seedance-compliance.md` 的过审规则（敏感年份模糊化、不出现敏感物件等）。
