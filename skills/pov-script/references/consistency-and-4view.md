# 角色一致性系统：角色基因卡 + 多年龄段 4 视图

适用于任意题材。对剧本里**每个反复出场的重要角色**都执行本系统，不只主角。一致性的锁定方式：**用 4 视图出角色设定图，作为图生视频的参考 / 首帧**，由图像保证跨镜形象统一。分镜里只用角色名 / 锚点指认是谁，**不再逐段描述外貌**（重复描述会与已定的角色图冲突）。

## 1. 角色基因卡（先输出，全片共用）

为每个重要角色定义跨年龄、跨场景不变的核心特征：

- 骨相 / 脸型：[如 鹅蛋脸、颧骨偏高、下颌线圆润]
- 眼睛：[形状、眼距、神态]
- 鼻型：[挺 / 塌、鼻翼宽窄]
- 嘴型 / 唇形：
- 肤色基调：[偏白 / 小麦 / 黝黑，冷调 / 暖调]
- 标志性识别点：[1-2 个跨龄不变的特征，如 右脸颊浅梨涡 / 虎牙 / 单眼皮 / 招风耳 / 浓眉；**避开** 痣 / 泪痣 / 疤 / 胎记 等审核敏感词]
- 神态气质：[如 内向沉静 / 倔强坚毅 / 温和憨厚 / 精明锐利]
- 体型 / 身形：[高矮、胖瘦、体态，如 偏瘦高、含胸、挺拔]

## 2. 多年龄段 4 视图：何时出

按《剧本解析》里每个角色的年龄跨度判定：

- 同一角色任意两个出场节点年龄相差 ≥ 5 岁 → 必须各出一张 4 视图
- 童年（≤12）/ 青少年（13-17）/ 青年（18-30）/ 中年（30-50）/ 老年（50+），每跨一档必出一张
- 同档内年龄接近的可共用一张，取中间年龄
- 角色全程年龄无显著变化 → 出 1 张当前年龄 4 视图即可

## 3. 4 视图输出格式（每张开头必复述基因卡核心特征）

```text
【4 视图 -【角色名】- [年龄段名] - [具体年龄]岁】

构图：左侧为人物头部大特写（Head Detail，头顶至肩部）——**必须标准正面、平视、正脸朝向镜头，不可侧脸、不可俯仰角**；右侧三视图为全身照（Front View 全身正面、Side View 全身正侧面、Back View 全身正背面，从头顶到脚底完整入镜，脚下留地面投影或薄灰）。**人物面无表情（中性脸）、双手自然下垂、自然直立站姿、不持任何道具**，白色或浅灰纯背景。三视图人物身高比例一致、机位高度一致（取人物胸高水平视角）、人物在画面中的占比一致。
人物核心特征（必须严格保持一致）：[复述第 1 节中该角色的 骨相/眼睛/鼻型/嘴型/肤色/标志性识别点/体型]
当前年龄：[X 岁]
当前服装：[贴合该年龄 + 年代 + 地域 + 身份的具体单品，颜色、材质、磨损细节、配饰]
当前发型：[随年龄合理变化，但发际线、发质保持一致]
当前神态：**面无表情、中性脸**（基础设定图，便于后续在分镜里驱动各种表情；此处不预设情绪）
姿态：**双手自然下垂、自然直立、不持道具**
道具：无——角色图一律不持道具、双手空置；关键道具改由《道具设定图》单独出图（见第 6 节）
质感：胶片颗粒、写实摄影、低饱和。
```

## 4. 跨图一致性硬约束（每张 4 视图末尾必加）

> "严格保持与本角色其他年龄段 4 视图相同的骨相、眼型、鼻型、嘴型、肤色基调、体型和标志性识别点（[具体复述]），仅服装、发型、神态随年龄合理变化。整体风格、光照、背景、构图模板与本角色其他 4 视图完全一致。"

## 5. 落到视频生成的一致性链路（推荐工作流）

1. 先用 4 视图提示词出**角色设定图**（文生图）。
2. 取设定图作为参考 / 首帧，用**图生视频（image-to-video）**做分镜，比纯文生视频的一致性强得多。
3. 分镜的"画面内容"字段只用角色名 / 锚点指认角色，**不重复描述外貌**——形象交给参考图 / 首帧，避免文字描述与已定角色图冲突。

## 6. 道具设定图（关键道具单独出图，不放在角色身上）

角色图一律双手空置；剧中贯穿的关键道具（信物、凶器、symbol 物件等）各自单独出一张设定图，便于后续在分镜里精准调用、保持道具一致：

```text
【道具图 -【道具名】】
构图：道具置于纯白 / 浅灰背景正中，单体特写，**画面里无人物、无手**；可给整体 + 1-2 个关键细节角度。
外观：材质、颜色、年代感、磨损 / 使用痕迹、尺寸比例。
质感：写实摄影、柔和顶光、浅景深、胶片颗粒。
```

## 7. 自检

- [ ] 每个重要角色都有基因卡，标志性识别点已避开敏感词
- [ ] 有显著年龄跨度的角色都有独立 4 视图，覆盖其全部出场年龄档
- [ ] 所有 4 视图右侧三视图均为全身照（头到脚完整），三张身高 / 机位 / 占比一致
- [ ] 左侧大特写为标准正面正脸、平视；人物面无表情、双手自然下垂、不持道具、纯背景
- [ ] 每张 4 视图都复述了同一份核心特征并附跨图硬约束
- [ ] 关键道具均已单独出《道具图》，角色图里不持任何道具
- [ ] 多角色之间特征区分明确，不会混淆
